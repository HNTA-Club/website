# HNTA Website

This repository hosts the Jekyll project for HNTA website. Based on Minimal Mistakes theme.

## Setting up the project

1. Install Ruby and Jekyll. Refer to <https://jekyllrb.com/docs/installation/> for instructions.
2. Clone the repository and navigate to the project directory.
3. Run `bundle install` to install the required gems.
4. Run `bundle exec jekyll serve` to start the development server. The website will be available at <http://localhost:4000>.
